HYPEBEAST에서 연재하는 업계 인사들의 Essential(가방 속 물건 등), Tech Essential(스마트폰 어플) 기사들에 모티브를 얻어 '나의 필수 iOS 어플들'을 정리해봤다.

어플들을 일-주-월 단위 별로 '1회 이상 실행하는가' 기준으로 분류하였고, 어플별 간략한 설명은 WHAT / WHEN / WHY / Paid or Free에 맞춰 간략히 코멘트(iPhone, iPad 구분 없이 정리).

 

Daily

+ Pocket Informant: 일정관리 / 일정 확인-등록 / Google Calendar-iCloud-iOS Reminders-Toodledo 간 연동 최고. 유료.

+ Mailbox: 메일관리 / 여러개 Gmail 계정 관리 / 'Inbox Zero'를 만들도록 도와주는 UX, 컨셉. 덕분에 매일 Inbox Zero의 희열(?)을 느낀다. 무료.

+ Evernote: 노트(Cloud 기반) / 미리 작성, 등록한 노트 보거나 메일로 재전송 할 때 / Cloud 기반 노트 서비스의 왕자. iOS에서는 거의 노트 보는 용도로만 사용. 무료.

+ Day One: 일기 / 지하철,버스,침대에서 간단히 일기 적을 때 / iOS와 Mac 용 Day One을 구매. UX도 좋고 Cloud 저장도 좋고. 확실히 일기를 더 쓰게 되는 것이 가장 좋고. 유료.

+ Solar: 날씨 / 아침 출근 전. 옷차림이나 우산 소지를 고민할 때 / Solar 만큼 '예쁜' 날씨 어플을 보지 못했음. 터치 스와이프로 시간대 별 날씨를 볼 수 있는 것도 좋고. 유료.

+ Feedly: RSS Reader / 수시로 / Google Reader가 떠난 빈 자리를 꿰찬 Feedly. iPad(iPhone, 웹 보다)에서 보기 참 좋다. 무료.

+ Pocket: Read Later / 수시로 / 많은 Read Later 서비스를 사용한 끝에 Pocket에 정착. 무료.

+ Cookie Words: 단어장 / 모르는 단어 찾아 볼 때 / 쿠키 단어장은 단어장이면서 사전이다. 블로그 리뷰도 썼지만 Peekaboo 기능은 최고. Evernote, Dropbox 연동도 좋고. 유료.

+ 스마트 월렛: 카드 정보 저장 / 포인트 카드 적립 할 때 / 바코드 기반 여러 멤버십 카드를 모바일로 담게 해주는 어플. CJ One, 롯데 멤버스, GS Point 등  편하게 사용 중. 무료.

+ 영웅문: 주식 거래 / 주식 매매 / 키움증권 HTS. 그 이상도 이하도 아님. 무료.

+ pooq: 공중파 TV 스트리밍-VOD / (공중파) 각종 예능-드라마 볼 때 / 대안이 없어서 pooq 사용 중. 덕분에(!) TV 프로 시청시간이 늘었다. 무료(서비스는 유료).

+ Tweetbot: 트위터 클라이언트 / 수시로 / 트위터 클라이언트의 왕중왕. 공식 트위터 앱, Osfoora, Tweetlogix 등 아무리 써보아도 iOS는 Tweetbot이 갑. 유료.

+ Instagram: 사진 기반 소셜미디어 / 수시로 / 유명하니 설명 스킵. 무료.

+ Cortex Camera:  카메라 / 멈춰있는 정물-풍경, 어두운 곳에서 사진을 고화질로 담고 싶을 때 / iPhone 카메라가 똑딱이를 위협하는 이유. 리뷰 참조. 유료.

+ Groove: 음악 재생 / 음악 들을 때 / Groove는 라이브러리 내 음악 재생목록을 Mix하는 특장점과 편한 UX로 어필. 유료.

 

Weekly

+ 1Password: 비밀번호 관리 / 웹 사이트 아이디 정보 생각 안날 때 / 로그인 계정, 비밀번호, 계좌, 각종 정보 등 1Password 없으면 어찌 기억할까 싶다. Mac 용도 구매. 유료.

+ 우리은행: 은행 / 이체 하거나 내역 조회 할 때 / 이 어플 덕에 Active X를 덜 만날 수 있어서 행복. 무료.

+ 코레일톡: 기차표 / 기차표 예매 /  코레일 홈페이지보다 어플이 100배 편하다. 무료.

+ CGV: 영화표 / 영화표 예매 / CGV의 '잘만든 어플' 때문에 다른 영화관 선택하기 어려워짐을 몸소 체험. 무료.

+ AVPlayerHD: 동영상 플레이어 / 미드, 영화를 iPad에 저장해서 볼 때 / 무인코딩 동영상 플레이어로 가장 유명. 속도, 기능 모두 우수. 유료.

+ yaPlayer: 동영상 플레이어 / 미드, 영화, 강의를 iPhone에 저장해서 볼 때 / AVPlayer와 유사한 기능. 속도, 기능 손색 없음. 유료.

+ AirVideo: 서버 동영상 스트리밍 / PC에 있는 동영상 복사하기 귀찮을 때 / 워낙 유명한 어플로 설명 스킵. 유/무료.

+ YouTube: 동영상 / (주로)뮤직비디오 찾아 볼 때 / 할 말 없음. 사랑해요 YouTube.

+ TEDiSubtitle: TED / TED 볼 때 / 저장, 자막 기능이 탁월. TED 어플 중 최고라고 생각. 유료.

+ Nike Running: 운동이력 기록 / 런닝할 때 / 런닝 시 위치 저장, 거리 저장, 속도 저장으로 운동 동기부여 제공. 유료.

+ Sleep Cycle: 수면 기록 및 분석 / 자기 전 / 아메리카노를 마셨거나, 피곤한 날 내가 깊은 잠을 잤는지, 아닌지를 알 수 있다. 유료.

+ T Map: 네비게이션 / 운전 시 / 내 차에 매립되어 있는 아이나비 네비게이션이 일 년 넘게 방치 중. T Map 사랑해요. 무료.

+ Aladin eBook: eBook / eBook 볼 때 / 알라딘 사용자라서 알라딘 eBook에 종속된 상태. 리디북스로 넘어갈까 고민 중. 무료.

+ Super 영단어숙어 50권: 영어 단어장 / 영단어 암기 / 여러 시험(토익, 토플, SAT, GMAT, GRE, IELTS 등)을 커버하는 단어장. 유료.

+ Trello: 생산성 / 볼 영화, 살 책 등 기록 할 때 / Trello는 iPad에서 그 UX가 극대화 된다. Trello 설명은 리뷰 참조. 무료.

 

Monthly

+ iPhoto: 사진 관리 / iPhoto 라이브러리 앨범, 이벤트 작성 및 수정 / 구매했을 때 기대만큼 자주 쓰고 있진 않음. Mac iPhoto과 연동만으로도 훌륭. 유료.

+ Momento: 소셜 미디어 이력 저장 / 월 1회 씩 / 내가 자취를 남긴 모든 소셜 미디어의 이력을 Momento로 저장. Lifelogging으로서 축적 중. 유료.

+ UPAD: PDF 필기 및 리더 / 논문, eBook에 필기 / iPad를 지참하는 날이 줄어들었지만, PDF에 필기하기에 UPAD가 가장 편리(iPad 필기 자체가 불편한건 함정). 유료.

+ ubrush: 그림 그리기 / iPad에 그림 그릴 때 / 그림에 소질이 없어서 거의 사용을 못하는 중. 그리기 좋아하는 지인들은 감탄하며 잘 써주는 어플. 유료.

+ TextGrabber: 문자 인식 OCR / 종이 책이나 기사를 찍어 텍스트로 바꿔야 할 때 (책 리뷰) / 한글 문자인식으로는 최고인 iOS 어플. 책 리뷰 때 간혹 이용.

 

 

정리하고 보니 남들의 Essential Apps.도 궁금하다.
