하루에도 웹 브라우저나 SNS 등에서 우리는 많은 기사나 블로그의 '링크'를 만나고, 볼거리/읽을 거리들을 접한다.

하지만, 버스나 지하철 안에서 브라우징 하는 동안에 만난 좋은 '링크'들을 모두 읽기에는 장소나, 스마트폰 화면의 가독성이 좋지 않기 때문에 놓치는 기사나 정보가 많다. 또 회사에서 웹 브라우저를 통해서 제목은 흥미가 있지만, 당장 기사나 블로그 포스트의 전문을 읽기에는 시간이 여의치 않을 때도 있다. 이럴 때, 나는 이러한 '링크'(정확히는 hyperlink의 웹 주소)를 쉽고 빠르게(터치 한 번, 클릭 한 번) 저장하고, 추후에 그 저장한 내용들을 보기 좋고, 쉽게 확인 할 수 있게 도와주는 서비스로 Instapaper를 사용하고 있다. 이번 포스트에서는 Instapaper의 편리하고 우수한 기능을 정리하였다.

 

Instapaper는 현재 웹 기반으로 http://instapaper.com 사이트와, 각종 스마트기기용 어플리케이션을 통해 iPhone, iPad, Android, Blackberry에서 서비스를 한다. Instapaper의 서비스 중 대표적 기능은 'Read Later'와 Readability 향상 기능이다.

 

 

Read Later

Read Later 기능이란 말 그대로 지금 보고 있는 화면의 내용(즉, 컨텐츠의 주소정보)를 나중에 보기위해 특정한 곳에 편하게 저장해주는 기능이다. Instapaper는 크게 두 가지 편의기능으로 이 Read Later를 지원하는데, 하나는 웹 브라우저에 Bookmarklet(즐겨찾기) 버튼 추가한 후, 이걸 누르면 Read Later에 저장이 되게 해주는 것이다. 웹 서핑 중에 그냥 막 클릭하면, Instapaper 내 계정의 공간에 차곡차곡 읽을 거리들의 목록이 저장 된다는 말이다. 단순하면서 아름답다.

 

아래 Bookmarklet은 사실 아래 코드처럼, 웹 브라우저가 띄운 현재 화면을 Instapaper로 보내서 내 계정에 저장하는 함수일 뿐이다. 하지만, 이 짧은 코드의 편의성을 왜 아직도 몰랐단 말인가! 이것도 instapaper.com 사이트에 가면 드래그&드랍으로 한방에 끝난다.

 

두번째는 iPhone, Android, iPad 등 가리지 않고 스마트 기기의 어플리케이션들 內 add-on 기능으로 사용할 수 있다는 것이다. 특별히 설치할 필요 없이, 트위터 하다가, 페이스북 하다가 읽을거리나 하이퍼링크를 Instapaper로 휙 저장할 수 있다는거다. 터치 한번으로. 개인적으로는 가장 애용하고, 훌륭하다고 생각하는 그들의 기술이고 컨셉이다. 연동이 되는 어플리케이션은 주요한 SNS, 잡지, 리더 어플리케이션 대부분(151개 어플리케이션)이고, 계속해서 늘어나고 있다(공개 developer api 덕분).

 

개인적으로 애용하는 pulse, tweetbot에서 Instapaper로의 연동 화면을 첨부했다.

 

그럼 이렇게 Read Later한 내용들은 어디에서 어떻게 볼 수 있는가, 웹은 Instapaper.com에서 등록한 계정(무료)로 로그인하면, 아래처럼 Read Later로 등록한 내용들이 보이게 된다.

 

iOS, Android, Blackberry 용 Instapaper 전용 어플리케이션도 있다. (iOS용 유료 등) 유료로 서비스를 제공하는 만큼 편의성이 뛰어나다. (동영상을 바로 볼 수 있고, 관리하는 방식, 공유하는 것 등...) 한 번 쯤 직접 사용해보길 추천하고 싶다.

 

 

Readability 향상 기능

사실, Instapaper는 Read Later 기능 만으로도 훌륭하다. 하지만, 대부분의 Read Later 되는 항목들이 'Article' 형태, 글의 형태라는 것에 집중해서 편의기능을 하나 더 추가해주었다. 가독성을 향상시켜주는 버튼을 웹 브라우저에 달아주는 방식과, Instapaper 웹이나 어플리케이션에서 Text 기능을 이용하는 방식이 있다.

SOPA 관련된 Article을 Read Later로 저장하였다. 실제 링크는 아래처럼, 많은 링크/광고/색깔 등으로 본문에 집중하기가 어렵다.

 

이러한 불편함을, 아래처럼 보기 좋고 깔끔하게 정제하여 보여준다. (글꼴크기, 줄간격, 세로단 간격, 글꼴변화까지...) 이 기능은 사실 이제는 Instapaper에만 있는 것은 아니다. (Apple Safari 최근 업데이트 때 Reader 기능을 추가해서, 이 기능과 같은 기능을 내재화 하였다.) 그렇지만, Instapaper의 Read Later 기능과 함께인 Text(Readability 향상) 기능은 막강함을 느낀다.

 

그 외 Instapaper의 기특한 점은.

  • 해당 Read Later 자료를 자신이 원하는 형태로 관리할 수 있다는 것 (Loved, Archive등을 통해)
  • PDF 등 파일 형태로 Export 해서 볼 수 있다는 점 (오프라인에 강점)
  • Print 할 때도 바로 연동 된다는 점 등이 있다.
